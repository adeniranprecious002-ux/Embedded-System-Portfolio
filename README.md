# Embedded Systems Portfolio

This repository serves as a central showcase of embedded systems and IoT projects built using **Arduino**, **ESP32**, **ESP8266**, sensors, actuators, wireless communication technologies, and real-world automation systems. Each project below links to its own dedicated repository containing source code, circuit diagrams, documentation, and implementation details.

---

## Featured Projects

### 1. Touchless Water Control System

A smart touchless water dispensing system designed to improve hygiene and reduce water wastage. The system automatically detects the presence of a user's hand and controls water flow without physical contact.

**Tech Stack:** Arduino/ESP32, Ultrasonic Sensor, Relay Module, Solenoid Valve, Embedded C++

🔗 https://github.com/adeniranprecious002-ux/touchless-water-control

---

### 2. ESP8266 LED Matrix Web Scroller

An IoT-based LED matrix display system that allows users to send custom scrolling messages through a web interface. Messages are transmitted wirelessly and displayed in real time on an LED matrix.

**Tech Stack:** ESP8266, MAX7219 LED Matrix, Wi-Fi, HTML/CSS, Embedded C++

🔗 https://github.com/adeniranprecious002-ux/ESP8266-LED-Matrix-Web-Scroller

---

### 3. Density and Sound-Based Traffic Light System

An intelligent traffic management system that dynamically controls traffic lights based on vehicle density and emergency vehicle siren detection, helping to reduce congestion and improve traffic flow.

**Tech Stack:** Arduino, Ultrasonic Sensors, Sound Sensor, LEDs, Embedded C++

🔗 https://github.com/adeniranprecious002-ux/Density-and-Sound-Based-Traffic-light

---

### 4. Automatic Fan Controller

A temperature-based automatic fan control system that adjusts fan operation according to ambient temperature, improving energy efficiency and user comfort.

**Tech Stack:** Arduino, Temperature Sensor, Relay Module, DC Fan, Embedded C++

🔗 https://github.com/adeniranprecious002-ux/automatic-fan-controller

---

### 5. Automatic Irrigation System

A smart irrigation solution that monitors soil moisture levels and automatically waters plants when necessary, helping to conserve water while maintaining healthy plant growth.

**Tech Stack:** Arduino/ESP32, Soil Moisture Sensor, Water Pump, Relay Module, Embedded C++

🔗 https://github.com/adeniranprecious002-ux/automatic-irrigation-system

---

### 6. Speed Measurement System for Road Safety Personnel

A vehicle speed monitoring system designed to assist road safety personnel in measuring and tracking vehicle speeds using sensor-based distance and timing calculations.

**Tech Stack:** Arduino, Infrared Sensors, LCD Display, Embedded C++

🔗 https://github.com/adeniranprecious002-ux/SPEED_MEASUREMENT_SYSTEM_FOR_ROAD_SAFETY_PERSONNEL
